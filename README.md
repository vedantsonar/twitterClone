# Twitter Clone - MERN

Twitter Clone made using MERN Stack

<img src="pic-1.png" alt="My Project Logo" width="400"/>


 ## Features

- Sign up/ Sign in via JWT Token
- Customize account
- Tweet
- Comment
- Retweet
- Follow user
- Like tweets and comments
- Edit tweets and comments
- Delete tweets and comments
